    Aide et bonnes pratiques

    Quelques conseils de base pour utiliser ce générateur de manière efficace :

    • Utilisez un mot de passe différent pour chaque service.
    • Privilégiez une longueur ≥ 16–20 caractères avec tous les types activés.
    • Ne stockez jamais vos mots de passe en clair dans un fichier texte synchronisé sans chiffrement.
    • Idéalement, combinez ce générateur avec un coffre-fort chiffré (KeePass, fichiers KDBX, etc.).
    • Évitez d’utiliser ce générateur sur une machine que vous ne contrôlez pas (poste public, PC d’hôtel, etc.).

    Ce site est purement statique : aucun mot de passe n’est envoyé sur un serveur. Tout se fait dans votre navigateur.
